Rocket Launch Tracker

This bot will notify the users in your server when a new rocket launch is announced.

Commands:

Command What does it do?
RT help Get the all the commands
RT track Get info about the recently added launch
RT get [number from 1 to 5] Get the latest added launches
RT start Setup the bot in a channel
RT stop Stop the bot from sending messages
